Project Overview

Nebula Strike is an action-packed, side-scrolling arcade space shooter designed for Android. Built from the ground up using Jetpack Compose, the game features smooth graphics, real-time collision detection, dynamic wave generation, immersive audio effects, settings configuration, and local high score tracking.

By utilizing modern Android components, Nebula Strike combines the nostalgic charm of retro arcade classics with clean, state-of-the-art programming paradigms, offering developers a solid reference for implementing custom Canvas-based games in Compose.

Architecture & Design

The project implements a decoupled Model-View-Controller design pattern suitable for interactive loop-driven frameworks.

GameEngine.kt Coroutines-based game loop, state transitions, physics updates
GameCanvas.kt Compose DrawScope rendering player, lasers, enemies, explosions
Entities.kt Data structures for Player, Enemy, Laser, PowerUp, Particle
SoundManager.kt Audio playback pooling using SoundPool & MediaPlayer APIs

The loop runs inside a Kotlin Coroutine synced to standard 60 FPS tick rates, computing physics offsets, boundaries, collision interactions, and score adjustments before notifying the drawing layer: 

// Standard update cycle in GameEngine.kt
fun updateGame(deltaTime: Float) {
    updatePlayer(deltaTime)
    updateLasers(deltaTime)
    updateEnemies(deltaTime)
    checkCollisions()
    cleanupEntities()
}
